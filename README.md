# 2048 Game

这是一个使用 React 和 TypeScript 实现的经典 2048 游戏。

## 功能特性

- 经典的 4x4 网格游戏玩法
- 键盘方向键控制
- 分数统计和最高分记录
- 游戏胜利（达到 2048）和失败（无法移动）检测
- 响应式设计，适配不同屏幕尺寸
- 美观的 UI 设计和动画效果

## 技术实现

- 使用 React 和 TypeScript 构建
- 核心游戏逻辑封装在 `Game2048` 类中
- 使用 Tailwind CSS 进行样式设计
- 使用 localStorage 存储最高分

## 项目结构

```
src/
├── components/
│   └── GameBoard/          # 游戏主界面组件
│       ├── index.tsx       # 游戏界面实现
├── Game2048.ts             # 2048 游戏核心逻辑类
├── App.tsx                 # 应用根组件
├── main.tsx                # 应用入口文件
└── index.css               # 全局样式
```

## 核心类说明

### Game2048 类

这是游戏的核心逻辑类，包含了所有游戏规则的实现：

- `initializeGame()`: 初始化游戏状态
- `move(direction)`: 处理四个方向的移动
- `addRandomTile()`: 在随机位置添加新数字块
- `checkGameStatus()`: 检查游戏状态（胜利/失败）
- `restart()`: 重新开始游戏

## 如何运行

1. 安装依赖：
   ```bash
   pnpm install
   ```

2. 启动开发服务器：
   ```bash
   pnpm run dev
   ```

3. 在浏览器中打开 http://localhost:5173 查看游戏

## 游戏玩法

1. 使用键盘上的方向键（↑ ↓ ← →）来移动数字块
2. 当两个相同数字的方块碰撞时，它们会合并成一个数字块
3. 每次移动后，会在空白位置随机生成一个数字块（90%概率为2，10%概率为4）
4. 目标是合成出2048数字块
5. 当棋盘被填满且无法再进行合并时，游戏结束

## 自定义

你可以通过修改 `tailwind.config.js` 文件来自定义颜色主题，或者修改 `Game2048.ts` 文件来调整游戏规则。
